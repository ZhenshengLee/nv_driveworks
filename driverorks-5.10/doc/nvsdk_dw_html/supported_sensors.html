Cameras Supported

This is a list of sensors the PDK packages and Driveworks supports out-of-the-box:

  • OnSemi AR0820 with Entron 30, 70, 120 degrees BM0A 24bit RGGB module, 8MP native resolution.
  • Sony IMX390 with 120 degrees RCCB/RGGB sensors and 200 degrees RGGB sensor.
  • Sony IMX623/IMX728 with RGGB/RCCB sensors.

Deprecated Camera Support

  • OnSemi AR0231 with Sekonix SF3324/SF3325 modules, 2MP resolution.
  • OnSemi AR0144.

Please refer to the SIPL Query Tool to double-check if the camera drivers are being detected correctly in your setup.

Any custom GMSL camera that is interface-able with the NvSIPL library, works with Driveworks if it is listed in the SIPL Query Tool output.
