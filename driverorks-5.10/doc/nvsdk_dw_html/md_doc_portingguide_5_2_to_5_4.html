This document will help you port your applications from DriveWorks SDK 5.2 to DriveWorks SDK 5.4.

Image Streamers

The cross process image streamer was deprecated in DW SDK 5.2 and is not available anymore. The dwChannel framework available as part of DW CGF should be used for cross process streaming. In process image streamers continue to work as before. Following API and types should not be used any more and will be removed in future release:

  • dwImageStreamer_initializeCrossProcess()
