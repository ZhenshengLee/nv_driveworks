◆ dwSensorLidarPlugin_parseDataBuffer

typedef dwStatus(* dwSensorLidarPlugin_parseDataBuffer) (dwLidarDecodedPacket *output, const dwTime_t hostTimeStamp, dwSensorPluginSensorHandle_t sensor)

Processes the data previously passed via the 'dwSensorPlugin_pushData' interface.

The interpreted memory buffer outputted from this API is owned by the plugin. The plugin shall support multiple buffers in flight via this API.

Parameters
[out]outputPointer to decoded lidar point cloud
[in]hostTimeStampSpecifies the host timeStamp when raw data was received.
[in]sensorSpecifies the sensor the data came from.
Returns
DW_INVALID_HANDLE - if the sensor handle is NULL or invalid
DW_NOT_AVAILABLE - if no frame is ready for consumption DW_SUCCESS

Send message to lidar sensor.

Parameters
[in]cmdIdentification of the message.
[in]dataRaw payload of the message.
[in]sizeRaw payload size of the message.
[in]sensorSpecifies the sensor to send the message data.
Returns
DW_INVALID_ARGUMENT - if the data is invalid
DW_NOT_SUPPORTED - if the sensor handle is NULL or invalid, or specified sensor doesn't support cmd or the output connection is not set
DW_FAILURE - if not able to send data out successfully
DW_SUCCESS - if send the message out to the sensor successfully
